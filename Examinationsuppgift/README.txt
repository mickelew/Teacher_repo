- Examinationsuppgift Biblioteket -

- Mål
Slutuppgift i "Programmering i Python" där målet är att
skapa ett program för bibliotek som för register över diverse media
samt deras värden.

- Installation
Inget specifikt behöver installeras för att köra programmet.

- Användning
Starta programmet genom att köra Bibliotek_2.py.
All input från användaren sker antingen i string-format eller integer-format,
aldrig båda samtidigt.
Filerna som behövs för att skapa ett register över den data användaren
registrerat skapas automatiskt när programmet avslutas.
